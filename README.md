# 🌐 Responsive Multi-Page Business Website

A modern, responsive, and multi-page business website built using HTML5, CSS3, and Vanilla JavaScript.

This project demonstrates the development of a complete company website with multiple pages, responsive navigation, reusable components, and a clean user interface.

---

## ✨ Features

- 📱 Fully Responsive Design
- 📄 Multi-Page Layout
- 🍔 Mobile Sidebar Navigation
- 🎨 Modern UI Design
- 📦 Products Page
- 🛠 Services Page
- ℹ️ About Page
- 📞 Contact Page
- 🖼 Image Gallery Sections
- 📍 Contact Information
- 📩 Contact Form
- 🌐 RTL (Right-to-Left) Support
- ⚡ Fast & Lightweight
